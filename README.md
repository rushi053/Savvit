# Savvit — Should you buy it now?

AI-powered purchase timing intelligence for iOS.

## Structure
- `ios/` — SwiftUI iOS app
- `server/` — Hono + TypeScript backend (Render)
- `docs/` — Product spec & architecture

## Stack
- **iOS**: SwiftUI, SwiftData, RevenueCat, PostHog
- **Backend**: Hono, Supabase, Perplexity Sonar, Keepa, Gemini Flash Lite
- **Deploy**: Render (backend), App Store (iOS)
